Features removed from the technique detector
=============================================

wah — swept bandpass filter (wah pedal). Indistinguishable from vocal
formant transitions at the mel spectrogram level. Fires on all
successive lyrics.

muted — lowpass-filtered timbre. Describes production technique, not a
perceptual event. CNN can't separate "muted guitar" from dull drum hits.

harmonics — isolated overtone series where the only differentiator is
timbral quality.

distortion — low-frequency signal is indistinguishable from bass guitar 
or any dense waveform. False positives on all bass content.

palm_mute — damped string attack. Same spectral signature as any short,
muffled percussive hit.

Rationale: Above features describe how a note is produced, not necessarily what happens musically.

Round 2 — spectral redundancy and neural relevance
===================================================

sforzando     — same spectral event as accent (sudden amplitude spike)
fortepiano    — compound of accent + decrescendo, not a distinct event
subito_forte  — same as accent
morendo       — slow decrescendo, redundant
marcato       — accent + staccato combined
tenuto        — indistinguishable from legato at spectrogram level
portato       — midpoint between staccato/legato, unlearnable boundary
spiccato      — spectrally identical to staccato
dead_note     — matches any percussive thud, too generic
tremolo       — fires on any rhythmic amplitude pattern
mordent       — too short (~40ms), overlaps grace_note
turn          — too similar to trill at mel spectrogram resolution
portamento    — slow glissando, redundant
rubato        — requires knowing "correct" timing, too context-dependent
flam          — ~25ms gap is ~1 spectrogram frame, unresolvable
ghost_note    — matches any quiet moment between louder hits
pizzicato     — matches every percussive attack with decay
hammer_on     — detecting absence of a new attack is unreliable
slide         — redundant with glissando
flutter       — tremolo + vibrato combined, both axes already covered

Rationale: For neural spike mapping, each label should correspond to a
distinct perceptual axis (amplitude change, pitch movement, temporal
pattern, expectation violation). Labels that are compounds of other
labels, spectrally indistinguishable from common musical events, or
below spectrogram resolution were cut. 41 -> 21 labels.
